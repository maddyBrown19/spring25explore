# Sketch of my mini project

**Goal:** Build a simple API and use it to display information in a basic UI. Backend practice (build an API that pulls from a database) and frontend practice (display the info from the API call in a dynamic UI).

**Timeline:** Three phases, spend about a week on each phase.

**Phase 1:** Fake data + UI prototype
- Make fake data (store in a list or CSV?)
- Make a simple UI that displays the fake data (all values will be hard coded at this stage)
- Use AI if it’s helpful? Maybe it can give me some boilerplate code that I customize

**Phase 2:** Backend research + mock up
- Research how to use AWS or Azure to host both the API and also the frontend
- Find a real dataset and set up an external database (if necessary?)
- Look through AWS tutorials and find one that's for starter API development
- Make a mock up of the API (very basic version, can execute through the command line or a short Python/Go script, pass in the raw request body in the call?)

**Phase 3:** Dynamic frontend supported by the API
- Edit the existing static frontend to make a call to the API and display the result from the API call where the static data was before
- Style the UI so it looks presentable
